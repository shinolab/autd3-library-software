Detailed Description

GainSequence provides a function to display Gain sequentially and periodically.

GainSequence uses a timer on the FPGA to ensure that Gain is precisely timed. GainSequence currently has the following three limitations.

  1. The maximum number of gains is autd::core::GAIN_SEQ_BUFFER_SIZE_MAX.
  2. The sampling interval of gains is an integer multiple of 25us and less than 25us x autd::core::SEQ_SAMPLING_FREQ_DIV_MAX.
