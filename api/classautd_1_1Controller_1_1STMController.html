void autd::Controller::STMController::start ( double  freq)

Start Spatio-Temporal Modulation.

Parameters
[in]freqFrequency of STM modulation

Generate STM modulation by switching gains appended by add_gain() at the freq. The accuracy depends on the computer, for example, about 1ms on Windows. Note that it is affected by interruptions, and so on.
